\subsubsection{Kompose}

Kompose ist ein Migrations-Tool, das speziell für den Wechsel von Compose zu Kubernetes entwickelt wurde. 
Kompose ist ein Mapper, der aus einer Compose-Datei Manifeste für jeden Service im Compose erstellt.
Zwar muss danach laut dem Unternehmen selber, die Manifeste etwas angepasst werden \parencite{kompose2026}, aber dennoch bietet das Tool den optimal Start für den Übergang.
Kompose ist ein offizeles Tool von Kubernetes selber.
